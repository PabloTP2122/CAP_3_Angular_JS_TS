Transiciones y animaciones

Transiciones

Para las transiciones se requiere

  1. La propiedad CSS que se quiere cambiar desde un estado inicial a un estado final.
    Pueden ser background-color, color, width...

    En este ejemplo es: 


                  trensition-property: background-color color;

  2. La duración. Es decir, el tiempo que demora en cambiar


                  
                transition-duration: 0.5s;

Además, se pueden tener las siguientes opciones para una transición más fina

  1. Transition timing function:

    • ease: (default) Inicia lento y luego va más rápido, finaliza lento
    • linear: El cambio ocurre con la misma tasa de cambio
    • ease-in: El cambio comienaza lento y va rápido antes del final
    • ease-out: El cambio comienaza rápido y va lento antes del final
    • ease-in-out: El cambio comienza lento, luego va rápido y al final es lento

                  transition-timing-function: ease;

  2. Transition delay


                  transition-delay: 2s;

CSS 2D transforms

Translate

  1. En este ejemplo la propiedad CSS que se va a modificar con la transición, sera transform y el tiempo de 2s, como se muestra abajo: 


                  
                transition: transform 2s

  2. En la pseudoclase :hover, se coloca la transformación que se requiere aplicar al colocar el cursor sobre la imagen.
    En este caso es una transformación del tipo translate
    Existen las siguientes transformaciones:
    • transform: translateX();
    • transform: translateY();
    • transform: translateZ();
    • transform: translate();


                  
                transform: translate(-500px, 200px);
