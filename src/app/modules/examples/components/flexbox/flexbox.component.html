Flexbox

Flexbox, o el modelo de caja flexible, es una técnica de diseño en CSS que permite crear páginas adaptables y eficientes.


Enfoque: Unidimensional. Se utiliza principalmente para diseñar una sola dimensión a la vez, ya sea una fila o una columna.

Uso: Ideal para la distribución y alineación de elementos dentro de un contenedor, especialmente cuando se necesita que los elementos se adapten dinámicamente al espacio disponible.

Ejemplos de uso: Menús de navegación, alineación de botones, distribución de elementos en un solo eje.
